# Game Game Go ROFL Verifier

MVP scope:

- verify versioned SquareXO replays;
- reject unknown games/rulesets;
- return deterministic result hashes;
- no wallet seed management;
- no wagering or payout logic.

Validation status in this repository: `STATIC_VALIDATION`. This folder has not
been built as a ROFL bundle or deployed to testnet in the current environment.
