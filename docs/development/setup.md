# Development Setup

## Requirements

- Python 3.10 or newer.
- Pygame dependency from `requirements.txt`.

## Install

```bash
python -m pip install -r requirements.txt
python -m pip install pytest pytest-cov ruff
```

## Run

```bash
python -m src.main
```

Local/offline mode is the default. No wallet, RPC endpoint or ROFL service is
required to launch the desktop app.
