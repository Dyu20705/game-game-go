# SquareXO

SquareXO is a local 1v1 Dots-and-Boxes-style game with deterministic replay
verification.

## Current Status

- Domain rules and replay encoding are unit-tested.
- Local Pygame session is enabled.
- Blockchain submission uses local/mock adapters by default.
- Online/testnet session is intentionally disabled in the MVP.

## Not Implemented

- No live Oasis deployment.
- No production ROFL bundle.
- No staking, wager settlement, token payout or NFT flow.
