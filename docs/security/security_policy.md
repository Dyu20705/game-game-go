# Security Policy

Game Game Go is pre-production desktop software. Please do not report security
issues through public exploit details before maintainers have had time to
respond.

Preferred reporting path:

- Use GitHub private vulnerability reporting if enabled for the repository.
- If private reporting is not available, open a minimal issue that says a
  security report is available without posting exploit details.

Never include private keys, seed phrases, wallet files, production RPC
credentials or deployment secrets in issues, pull requests or logs.

Current blockchain/Oasis work is a local/pre-testnet scaffold and has not been
externally audited or deployed.
